  6. 您參與的MAKE UP FOR EVER活動
  7. 特定健康資料，若您知會我們因使用我們的任何產品（僅適用於MAKE UP FOR EVER）而導致的任何不良副作用

若您的聯絡方式有任何變更，請及時以書面通知我們。

個人資料的儲存期限

我們僅出於滿足您的需求或根據我們的法律或商業義務之目的儲存您的個人資料，且僅會在為達成這一目的之所需期限內儲存您的個人資料。

一般情況下，您的個人資料在我們的資料庫中的保存期限為：

  1. 如果您為「客戶」（即您已經購買了一款MAKE UP FOR EVER產品）：作為客戶關係管理之用，您個人資料的儲存期限為首次惠顧後起計不超過10年，但作為業務發展之用，則儲存期限為首次惠顧後起計不超過5年；該期限將在您每次與MAKE UP FOR EVER展開互動（如惠顧）後更新
  2. 如果您為「潛在客戶」（即您從未購買過MAKE UP FOR EVER產品，但對MAKE UP FOR EVER品牌很感興趣）：在收集您的個人資料後儲存不超過3年；該期限將在您每次與MAKE UP FOR EVER展開互動（如參加活動）後更新
  3. 如果您與公共關係部門保持聯繫：在您首次接觸後儲存10年；該期限將在您每次與MAKE UP FOR EVER展開互動（如要求獲得與MAKE UP FOR EVER相關的資訊）後更新

如果您的電腦安裝了Cookie，其儲存期限將不會超過13個月。

如果我們不再需要使用您的個人資料，我們將從系統和記錄中將其刪除或匿名，由此無法再從中識別您的身份。

個人資料的保護和保密方式

我們已採取合理及適當的預防措施，使您的資料免於損失、濫用、篡改、毀壞或被未經授權的第三方存取。

我們透過何種方式履行跨境資料保護？

由於MAKE UP FOR EVER在全球多國均有業務營運，所以您的某些資料可能會由您居住國家以外的地方收集、存取或儲存。 您須知道各地的資料保護和安全要求可能有所不同，所以您的資料可能無法得到與您居住國家同等程度的保護。 然而，MAKE UP FOR EVER和我們集團旗下的公司已採取保護措施，以確保不論您的資料位於何地，均得到適度的保護，例如使用歐盟（被視為擁有全球最全面的資料保障法律）認可的資料傳輸方式。 我們亦要求代替我們接收個人資料的第三方合作夥伴，履行相關的適用資料傳輸義務。

我們為某些地區居民提供的額外保障

MAKE UP FOR EVER已為涵蓋範圍內的所有個人，採取本私隱政策中提及的資料保護和安全措施。 另外，我們亦已根據地方或區域要求，識別並採取資料保護和安全措施。 而採取的措施包括歐盟/歐洲經濟區居民必須遵守的《一般資料保護規範》（簡稱GDPR；歐盟法規編號：2016/679）。

GDPR賦予資料當事人以下權利：

  1. 知情權：您有權以清晰透明和通俗易懂的方式了解我們如何使用您的個人資料以及您所擁有的個人權利。您可在本政策中了解所有此等資訊
  2. 存取權：您有權存取MAKE UP FOR EVER所擁有的與您相關的個人資料
  3. 更改權：如果您的個人資料不正確或未更新，您有權更改；同時，如果您的個人資料不完整，您有權提供完整資料
  4. 刪除權/被遺忘權：您有權刪除或清除您的個人資料。請注意，這並非一項絕對權利，因為我們可能擁有保留您個人資料的法律或合法依據
  5. 對直接市場推廣的反對權：您可隨時取消訂閱或選擇退出我們的直接市場推廣通訊。您可以點擊我們向您發送的任何電子郵件中的專用連結來完成此項操作。您亦可以要求接收關於我們的產品與服務的非個人化通訊
  6. 隨時撤銷根據個人意願進行資料處理的權利：如果有關的資料處理事宜需取得您的同意方能進行，您可撤銷同意。
  7. 資料移轉的權利：您有權從我們的資料庫將資料移動、複製或轉移至另一資料庫。此項權利僅適用於您已提供之資料，其處理流程以合約或您的同意為基礎，並通過自動化方式進行處理。

您亦有權決定在不幸身故或遇上外來的種種情況時，您的個人資料將執行的保護指引和指令。

您亦有權聯絡您所在國家/地區的資料保護機構，以就MAKE UP FOR EVER的資料保護和私隱政策提出申訴。

如果您希望行使上述權利，或您對我們如何處理您的個人資料有任何疑問或投訴，請根據以下方式聯絡我們。

在處理您的請求前，我們可能會要求您提交身份證明資料以及與您的請求相關的詳細資料

其旨在讓您了解在您瀏覽本網站時我們處理的瀏覽資訊來源和用途，以及您的相關權利。

 

甚麼是COOKIE？

瀏覽我們的網站時，在您選擇啟用Cookie的情況下，我們可能會通過您的網絡瀏覽器，在您的設備（電腦、智能手機、平板電腦）上儲存一個文字檔。

此文字檔即是Cookie。MAKE UP FOR EVER可在其有效期或儲存期限內，於您下次瀏覽時識別您的設備。

只有Cookie的發出者可以讀取或修改Cookie中儲存的資料。

您可參閱下方關於由MAKE UP FOR EVER或第三方儲存於您的設備上的Cookie資訊，以及在您的設備上刪除/停用Cookie的方法。

 

MAKEUPFOREVERHK.SHOP網站發出的COOKIE有何用途？

Cookie有多種類型，當中部分由MAKE UP FOR EVER及其供應商直接發出，亦有一些由第三方公司發出。

  1. MAKE UP FOR EVER及其供應商發出的Cookie

您瀏覽我們的網站時，可能會有不同類型的Cookie儲存於您的設備上。

必要型Cookie是瀏覽我們網站必不可少的Cookie，可確保訂單流程順暢執行。刪除此類Cookie會導致瀏覽故障，以致無法提交訂單。此類Cookie可能會由MAKE UP FOR EVER或其供應商儲存於您的設備上。

分析型和個人化Cookie並非瀏覽網站所必需的Cookie，但能夠方便搜尋，提升您的購物體驗。

廣告型Cookie亦非瀏覽網站所必需的Cookie，但能夠為您提供更合適的廣告。

如果您的設備有多人使用，以及在同一設備上設有多個網絡瀏覽器，我們無法確保針對您設備的服務和廣告僅配合您本人對設備的使用習慣，而非他人對此設備的設置。在此情況下，您須自行決定是否與他人共用設備和您的網絡瀏覽器Cookie設定配置，並須承擔有關責任。

  1. 第三方公司發出的Cookie

第三方在本網站上發出和使用Cookie，須遵守第三方的私隱政策。這些並非瀏覽本網站所必需的Cookie。

我們可能會在本網站的第三方應用程式中包含Cookie，以便您與他人分享本網站的內容，或讓他人知悉您曾瀏覽過的網站內容或您對本網站內容的看法。Facebook、Twitter、Instagram、Pinterest等社交網絡的「分享」、「讚好」按鈕亦有此類作用。

提供此類應用按鈕的社交網絡，可通過此類按鈕識別您的身份，即使您在瀏覽本網站時並未使用此類按鈕。社交網絡收集有關您瀏覽本網站的資訊及與其所持有的個人資料相關的資訊所採用的流程不受我們控制。請查閱此等社交網站的私隱政策。

 

管理儲存於您的設備上的COOKIE

透過點擊此處、Cookie橫額或makeupforeverhk.shop 頁腳，您可以決定是否透過Cookie設定工具將Cookie儲存在您的設備上。

您亦可按照意願自行設置網絡瀏覽器，在您的設備上接受及儲存Cookie，或停用Cookie。

每個網絡瀏覽器的配置各有不同。您的網絡瀏覽器的輔助說明選單中通常載有相關描述。請前往查閱配置您的瀏覽器之方法，了解如何變更您的Cookie設定。

  1. Internet Explorer™用戶：https://support.microsoft.com/en-us/help/17442/windows-internet-explorer-delete-manage-cookies
  2. Safari用戶：https://support.apple.com/kb/PH19214?locale=fr_FR&viewlocale=en_US
  3. Chrome用戶：https://support.google.com/chrome/answer/95647?hl=en&hlrm=en
  4. Firefox用戶：https://support.mozilla.org/en-US/kb/enable-and-disable-cookies-website-preferences
  5. Opera™用戶：http://help.opera.com/Windows/10.20/en/cookies.html

請注意：您在網絡瀏覽器上針對接受或停用Cookie所作出的設定，將會改變您的互聯網瀏覽體驗，並影響部分需要使用此等Cookie方可享用的服務。 例如，如果停用必要型Cookie，您可能無法再在本網站上提交訂單。

 

看到因應您的瀏覽喜好而提供的特別優惠和廣告有何好處？

此項功能的目的旨在向您顯示最為相關的優惠和廣告。有見及此，Cookie技術會根據您瀏覽本網站的記錄進行推斷，使我們能夠即時向您展示您最感興趣的內容。

您透過您的設備瀏覽網站時，您表示感興趣的內容，會決定本網站的廣告資源，使其能夠向使用者提供服務，而有關服務通常都是免費的。無可置疑，您會寧願看到切合您需要的優惠和廣告，而非您完全不感興趣的內容。同樣，MAKE UP FOR EVER和廣告商都希望其提供的優惠和廣告，能夠向感興趣的網絡使用者顯示。
